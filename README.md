# Bibliotheca

Bibliotheca é um projeto aberto, criado com foco em estudo, experimentação e compartilhamento de conhecimento. Ele foi pensado como uma base simples, legível e maleável , algo que outras pessoas possam ler, modificar e aprender, sem que isso signifique apagar a autoria original.

Este repositório existe tanto como produto final quanto como material didático.

---

## Filosofia do projeto

Este projeto é **open source**, mas não é domínio público.

Isso significa:
- Você pode estudar o código livremente.
- Você pode modificar o código para uso pessoal ou educacional.
- Você pode criar sua própria versão **editada** do projeto.

O que não é permitido:
- Revender este projeto ou partes dele como produto próprio.
- Copiar o código e redistribuí-lo sem modificações relevantes.
- Remover créditos e apresentar o código como se fosse totalmente seu.

A ideia aqui não é travar aprendizado , é impedir apropriação preguiçosa.
